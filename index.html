HOW IT WORKS

完整使用流程

首次部署約需 5 分鐘，之後每次出團只需 1–2 分鐘

1

部署到 GitHub Pages（一次性）

在 GitHub 建立 public repo，命名 lefun-ebook，上傳所有檔案後至 Settings → Pages 啟用。

僅需一次
2

設定 API Keys（首次使用）

在 tool.html 右上角點「⚙ 設定 API Keys」，輸入 Anthropic API Key 及 GitHub Token。Keys 存於本機 localStorage，不上傳任何伺服器。

存入瀏覽器後免重設
3

上傳 PDF 填寫資訊

每次出團：開啟 tool.html，上傳行前說明 PDF，填寫電子書 ID（如 20260509-jeju）、團名、日期、領隊等基本資訊。

4

一鍵產生 & 分享

按下「一鍵產生電子書」，AI 解析完畢後自動發佈，複製分享連結傳給遊客即可。連結格式：

https://[帳號].github.io/lefun-ebook/viewer.html?id=[電子書ID]
SETUP

取得 API Keys

只需兩個 Key，設定一次即可長期使用

🤖 Anthropic API Key

1
前往 console.anthropic.com
2
登入或建立帳號
3
左側選單 → API Keys → Create Key
4
複製 sk-ant-... 格式的金鑰

🐙 GitHub Personal Access Token

1
前往 github.com/settings/tokens
2
Generate new token (classic)
3
勾選 repo 完整權限
4
複製 ghp_... 格式的 Token（只顯示一次）
